사이버 범죄
데이터 대시보드

사이버 범죄의 유형과 연도별 흐름을 인터랙티브하게 분석하는 데이터 시각화 서비스

경찰청 공개 데이터를 기반으로 금융범죄, 사기, 해킹, 성폭력, 도박 등 주요 사이버 범죄의 발생건수와 검거건수를 분석했습니다. Pandas로 데이터를 전처리하고 Streamlit과 Matplotlib·Plotly를 활용해 사용자가 범죄 유형과 분석 방식을 선택할 수 있는 대시보드로 구현했습니다.

01. OVERVIEW

프로젝트 개요

어떤 서비스인가요?

사이버 범죄 통계를 한눈에 보는 분석 대시보드

연도별·범죄 유형별 발생건수와 검거건수를 비교하고, 범죄 증가율과 주요 통계까지 확인할 수 있는 데이터 분석 서비스입니다.

왜 만들었나요?

복잡한 범죄 통계를 쉽게 이해하도록 만들기 위해

공개 통계는 표 형태로 제공돼 흐름을 파악하기 어려웠습니다. 이를 그래프와 요약 지표로 변환해 직관적으로 확인하도록 구현했습니다.

누구를 위한 서비스인가요?

사이버 범죄 동향을 확인하는 사용자와 분석 담당자

특정 범죄의 발생 추이와 검거 현황, 유형별 비중을 빠르게 비교하고 싶은 사용자를 대상으로 구성했습니다.

02. MY ROLE

담당 역할

팀장으로서 일정과 업무를 조율하고, 데이터 전처리부터 Streamlit UI와 시각화 기능까지 담당했습니다.

다양한 범죄 데이터를 연도와 범죄 유형 기준으로 통합하고, 사용자가 분석 옵션을 선택하면 차트가 동적으로 변경되도록 구현했습니다. 또한 발표 과정에서 주요 분석 결과와 범죄 증가 흐름을 설명했습니다.

02

05

05. FEATURES

핵심 기능

FEATURE 01

사이버 범죄 현황 대시보드

  • 총 발생건수 및 연평균 통계
  • 전체 증가율과 최대 발생 연도

FEATURE 02

  • 유형별 비중 계산
  • 상반기 범죄 통계 제공
  • 주요 범죄 유형 비교
FEATURE 03

범죄 유형별 발생·검거 추이 분석

사용자가 피싱 등 세부 범죄를 선택하면 연도별 발생건수와 검거건수를 선 그래프로 비교하고, 다양한 분석 옵션을 선택할 수 있습니다.

  • 범죄 유형 선택
  • 2018~2023 연도별 추이
  • 발생건수와 검거건수 비교
  • 전체 추이·검거율·비중 분석

06. TROUBLESHOOTING

문제 해결 경험

01

연도별 범죄 분류 기준이 달라 데이터를 바로 비교할 수 없었던 문제

문제

공개 데이터마다 범죄 유형명과 분류 기준이 달라 연도별 수치를 단순 병합할 수 없었습니다.

원인

일부 연도에서 범죄 명칭이 바뀌거나 세부 범죄가 상위 유형으로 통합되어 있었습니다.

해결

공통 분류 기준을 정의하고 유사한 범죄 유형을 매핑하는 정규화 기준을 만들었습니다.

결과

여러 연도의 범죄 데이터를 하나의 기준으로 비교하고 추이 그래프를 생성할 수 있었습니다.

02

대용량 데이터를 반복 처리하면서 화면 응답이 느려진 문제

문제

사용자가 옵션을 변경할 때마다 전체 데이터를 다시 처리해 대시보드 반응 속도가 느려졌습니다.

원인

필터 변경마다 CSV 로드, 정제와 집계 연산을 처음부터 다시 수행했습니다.

해결

전처리 결과를 재사용하고, 필요한 열과 범죄 유형만 먼저 필터링하도록 구조를 개선했습니다.

결과

분석 옵션 변경 시 차트가 더 빠르게 갱신되도록 개선했습니다.

03

운영체제에 따라 한글 그래프가 깨지는 문제

문제

macOS와 Windows에서 Matplotlib 차트의 한글 제목과 축 이름이 사각형으로 표시됐습니다.

원인

각 운영체제에서 사용하는 기본 한글 폰트가 달랐습니다.

해결

macOS는 AppleGothic, Windows는 Malgun Gothic을 적용하도록 분기 처리했습니다.

결과

팀원별 개발 환경에서도 한글 그래프가 정상적으로 표시되도록 수정했습니다.

04

범죄 유형 선택과 차트 데이터가 일치하지 않았던 문제

문제

특정 범죄를 선택해도 다른 유형의 수치가 차트에 표시되는 경우가 있었습니다.

원인

화면 표시명과 데이터프레임 열 이름이 달라 잘못된 컬럼을 참조했습니다.

해결

사용자 표시명과 실제 데이터 컬럼을 연결하는 매핑 딕셔너리를 만들어 필터 로직을 통일했습니다.

결과

선택한 범죄 유형에 맞는 통계와 차트가 정확하게 표시되도록 개선했습니다.
